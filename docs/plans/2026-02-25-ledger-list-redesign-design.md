# 台账列表界面重构设计方案

**日期**: 2026-02-25  
**页面**: templates/valves/list.html  
**目标**: 扩大表格空间占比，紧凑化工具栏布局

---

## 一、当前问题

1. 工具栏占用空间较大，包含搜索框、状态筛选、批量操作按钮、新增按钮、集合详情按钮等
2. 表格区域有 `max-height: calc(100vh - 280px)` 限制，未充分利用垂直空间
3. 页面整体间距较大，表格空间占比约70%

---

## 二、设计方案

### 2.1 工具栏重构

| 元素 | 当前 | 调整后 |
|------|------|--------|
| 搜索框宽度 | 220px | 180px |
| 搜索框高度 | 40px | 36px |
| 下拉框高度 | 40px | 36px |
| 按钮内边距 | 0 20px | 0 16px |
| 工具栏间距 | gap: 12px | gap: 8px |
| 工具栏内边距 | padding: 12px | padding: 8px 12px |

### 2.2 表格区域

- 移除 `max-height: calc(100vh - 280px)` 限制
- 改为 `height: calc(100vh - 工具栏高度 - 分页高度)`
- 表格内部可独立滚动

### 2.3 分页区域

- 精简显示：只保留"共X条记录，第X/Y页"和上一页/下一页按钮
- 移除中间页码列表

---

## 三、预期效果

- 表格空间占比从约70%提升至85%以上
- 工具栏从原来的约80px高度缩减至约60px
- 页面整体更加紧凑，数据展示更高效
